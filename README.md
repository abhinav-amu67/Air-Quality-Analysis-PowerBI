# Air Quality Analysis Dashboard (Power BI)

## Project Overview
This project analyzes air quality data across different cities and states in India using Microsoft Power BI.

The objective is to identify pollution patterns, analyze AQI trends, and highlight the most polluted cities through an interactive dashboard.

## Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset

## Dashboard Features
• Average, Maximum, and Minimum AQI indicators  
• AQI trend over time  
• Top 10 most polluted cities  
• State-wise AQI comparison  
• AQI category distribution  
• City-wise AQI map visualization  
• Interactive filters for City, State, and Date  

## Dataset Source
National Data & Analytics Platform (NDAP)  
https://ndap.niti.gov.in
